Lista de Tareas (To-Do List) en React

Descripción breve:
-Este proyecto es una aplicación de lista de tareas (To-Do List) desarrollada con React. Permite agregar, eliminar y marcar tareas como completadas.
-Además, incluye funcionalidades adicionales como búsqueda de tareas, paginación, y almacenamiento local en el navegador.
-Se utilizan iconos de react-icons para una interfaz más atractiva.

Características:
-Agregar nuevas tareas.
-Eliminar tareas.
-Marcar tareas como completadas.
-Búsqueda de tareas.
-Paginación para manejar grandes listas de tareas.
-Almacenamiento local de tareas en el navegador.

Requisitos previos:
Antes de ejecutar el proyecto, asegúrate de tener instalado lo siguiente en tu máquina: Node.js: https://nodejs.org/en

Instrucciones de instalación

1. Clona este repositorio en tu máquina local utilizando bash:

-Ejemplo: git clone https://github.com/CursoFullStackAgosto/xxxxxxxxxxx.git

2. Instalar dependencias:

Dentro del directorio del proyecto, instala las dependencias necesarias ejecutando desde la terminal:
npm install
npm install react-icons

3. Ejecutar el proyecto
Para iniciar la aplicación en modo de desarrollo, ejecuta desde la terminal:

npm run dev

Este comando nos dará un link para poder acceder a la pagina creada. 
Este link se debe colocar en el navegador o presionar ctrl + click

Ejemplo http://localhost:5173/
